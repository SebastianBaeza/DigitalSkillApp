## Correr el código

### Frontend:
Antes de correr el frontend, debe situarse en el directorio "react-app" y ejecutar los siguientes comandos:

```npm install```

Luego, para correr el frontEnd se necesita usar el siguiente comando en la terminal:

```npm run dev```
